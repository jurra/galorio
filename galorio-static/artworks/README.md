# Sample Artwork Images

This folder contains placeholder SVG images for demonstration purposes. 

## To use your own artwork:

1. Replace these placeholder files with your actual artwork images
2. Keep the same filenames or update the CSV configuration accordingly
3. Supported formats: JPG, JPEG, PNG, GIF, WebP
4. Recommended size: At least 1200px on the longest side for best quality

## Current placeholders:

- `moonlight-reflection.jpg` - Featured landscape piece
- `feather-canyons.jpg` - Vibrant canyon artwork
- `coastal-breeze.jpg` - Peaceful ocean scene
- `urban-symphony.jpg` - Featured cityscape
- `abstract-form.jpg` - Bold abstract piece (marked as sold)
- `forest-whispers.jpg` - Gentle watercolor
- `profile.jpg` - Artist profile photo

Each image has a corresponding `.json` metadata file with artwork details.
